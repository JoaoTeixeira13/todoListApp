
# ✅  To Do List App

## Live Page

https://radiant-badlands-37206.herokuapp.com/

##  ✅  Overview

This is a single-page application where users can manage a To-Do list.

## ✅  Features

-   Users can add tasks to the list, and delete them by clicking on the correspondent checkbox.
-   Data saved in MongoDB Atlas.
